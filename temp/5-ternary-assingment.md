# Там, где возможно, в присвоении значения вместо if используется тернарный оператор

Неправильно:

```js
let sex;
if (male) {
  sex = 'Мужчина';
} else {
  sex = 'Женщина';
}
```

Правильно:

```js
const sex = male ? 'Мужчина' : 'Женщина';
```
